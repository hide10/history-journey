# 138億年の旅 — 宇宙のはじまりから、あなたまで

歴史が苦手な人のための、歴史入門ウェブサイトです。

ビッグバンから、地球の誕生、生命、人類、世界史、日本史、そして「今のあなた」まで――
138億年をひとつの物語として、スクロールで旅する体験型ページになっています。

**公開ページ:** https://hide10.github.io/history-journey/

## 特徴

- **宇宙カレンダー** — 138億年を「1年」に圧縮すると、人類の歴史は大晦日の最後の数十秒
- **章ごとに変わる星空** — スクロールすると空の色がゆっくり変化
- **年号より「へぇ！」** — 世界史・日本史はタップで開く豆知識カード形式
- スマホ・PC両対応、依存ライブラリなしの静的サイト（HTML/CSS/JS のみ）

## 構成

```
index.html      … 本体（全章のマークアップ）
css/style.css   … スタイル
js/data.js      … 世界史・日本史・宇宙カレンダーのデータ
js/main.js      … 星空Canvas・スクロール演出・タイムライン描画
```

## ローカルで見る

```bash
python3 -m http.server 8000
# → http://localhost:8000
```
