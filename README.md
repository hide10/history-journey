# 138億年の旅 — 宇宙のはじまりから、あなたまで

歴史が苦手な人のための、歴史入門ウェブサイトです。

ビッグバンから、地球の誕生、生命、人類、世界史、日本史、そして「今のあなた」まで――
138億年をひとつの物語として、スクロールで旅する体験型ページになっています。

**公開ページ:** https://hide10.github.io/history-journey/

## 特徴

- **シネマティックな章扉** — 各章の世界観をAI生成のドキュメンタリー風写真で演出
- **宇宙カレンダー2.0** — 138億年を「1年」に圧縮する3段ズーム図解（12ヶ月グリッド→24時間円環→最後の8分）
- **わかる図解** — 地球24時間時計・生命の系統樹・人類拡散マップ・文明並行年表・時代の長さ比べ・人口カーブをSVGで自作
- **本物を見てみよう** — ハッブル、地球の出、ラスコー壁画、源氏物語絵巻など実物資料を出典付きで掲載
- **章ごとに変わる星空** — スクロールすると空の色がゆっくり変化
- **年号より「へぇ！」** — 世界史・日本史はタップで開く豆知識カード形式
- スマホ・PC両対応、依存ライブラリなしの静的サイト（HTML/CSS/JS のみ）

## 構成

```
index.html       … 本体（全章のマークアップ）
css/style.css    … スタイル
js/data.js       … 各章のテキスト・図解データ・資料メタデータ
js/calendar.js   … 宇宙カレンダー2.0（SVG）
js/diagrams.js   … 各章のSVG図解6本
js/main.js       … 星空Canvas・スクロール演出・タイムライン・資料カード描画
tools/gen_art.py … 章扉絵の生成スクリプト（ComfyUI API）
assets/img/      … 章扉絵（AI生成 / WebP）
assets/artifacts/… 実物資料画像（下記クレジット参照）
```

## ローカルで見る

```bash
python3 -m http.server 8000
# → http://localhost:8000
```
